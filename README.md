# nerv-sh/homebrew-tap

Homebrew tap for [Nerv](https://github.com/nerv-sh/nerv) — IDE-grade
autocomplete for your terminal. macOS (Apple Silicon) + zsh.

```sh
brew install nerv-sh/tap/nerv
eval "$(nerv init zsh)"
```

`Formula/nerv.rb` is updated automatically on every
[release](https://github.com/nerv-sh/nerv/releases) by the
`homebrew-bump` workflow. Issues → [nerv-sh/nerv](https://github.com/nerv-sh/nerv/issues).
